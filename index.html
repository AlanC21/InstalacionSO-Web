Instalación de Sistema Operativo

1. Preparativos

  1. Verificación de Requisitos: Asegúrate de que tu hardware cumple con los requisitos mínimos para la versión de Windows que planeas instalar. Por ejemplo, para Windows 11, necesitarás un procesador de 1GHz o más rápido con 2 o más núcleos, 4GB de RAM, 64GB de almacenamiento y una tarjeta gráfica compatible.
  2. Conexión de Periféricos: Conecta un teclado y mouse USB a tu notebook si no están integrados. Verifica que ambos funcionen correctamente.
  3. Descarga de la Imagen ISO: Descarga la imagen ISO de Windows desde el sitio oficial de Microsoft. Asegúrate de elegir la edición correcta (Home, Pro, etc.).
  4. Preparación de la Unidad USB: Obtén una unidad USB con al menos 8GB de capacidad. Se recomienda usar una unidad nueva o formatearla previamente.
  5. Descarga de Software de Creación de USB Booteable: Descarga e instala un programa como Rufus, que es gratuito y compatible con Windows, para crear una unidad USB booteable.

2. Creación del USB Booteable

  1. Inicio del Programa: Abre Rufus una vez instalado.
  2. Selección del Dispositivo: En la opción "Dispositivo", selecciona tu unidad USB. Haz clic en "Seleccionar" y elige la imagen ISO de Windows que descargaste.
  3. Configuración del Esquema de Partición: Elige "Tabla de Partición GPT" para unidades de 4GB o más.
  4. Formato de la Unidad: Selecciona "NTFS" como el sistema de archivos.
  5. Creación del USB Booteable: Revisa la configuración y haz clic en "Comenzar". Rufus te mostrará una advertencia, haz clic en "Aceptar" para continuar. El proceso puede tardar varios minutos. Una vez finalizado, verás el mensaje "¡Listo!".

3. Configuración del BIOS/UEFI

  1. Acceso al BIOS/UEFI: Reinicia tu notebook y presiona repetidamente la tecla de acceso al BIOS/UEFI (por ejemplo, F2, Del, F10 o Esc).
  2. Modificación del Orden de Arranque: Dentro del BIOS/UEFI, cambia el orden de arranque para que la USB sea la primera opción. Habilita el modo Legacy o UEFI según sea necesario.
  3. Guardado de Cambios y Reinicio: Guarda los cambios y reinicia para arrancar desde la USB.

4. Instalación de Windows

  1. Arranque desde USB:Si configuraste correctamente el BIOS/UEFI, tu notebook arrancará desde el USB y verás un mensaje como "Press any key to boot from USB".
  2. Inicio de la Instalación:Aparecerá la pantalla de carga de Windows. Selecciona tu idioma, formato de hora y método de entrada de teclado. Luego, haz clic en "Siguiente" y "Instalar ahora".
  3. Clave de Producto:Si tienes una clave, ingrésala. Si no, selecciona "No tengo clave de producto".
  4. Selección de Edición:Elige la edición de Windows que deseas instalar.
  5. Aceptación de Términos de Licencia:Lee y acepta los términos de licencia.
  6. Tipo de Instalación:Selecciona "Personalizada: Instalar solo Windows (avanzado)".
  7. Selección y Partición del Disco:Selecciona el disco donde instalarás Windows. Si es necesario, crea nuevas particiones o formatea las existentes. Luego, haz clic en "Siguiente".
  8. Proceso de Instalación:Windows comenzará a copiar archivos e instalarse. Tu notebook se reiniciará varias veces durante este proceso.
  9. Configuración Inicial:Elige tu región, disposición de teclado, configura tu cuenta de usuario y ajusta las opciones de privacidad.
  10. Finalización de la Instalación:Windows finalizará la configuración y te llevará al escritorio.
  11. Instalación de Controladores:Visita la página del fabricante de tu notebook y descarga los controladores más recientes. Instálalos uno por uno, reiniciando cuando sea necesario.

5. Configuración y Optimización Posterior a la Instalación

  1. Actualizar Windows:Ve a Configuración > Actualización y seguridad > Windows Update > Buscar actualizaciones. Instala las actualizaciones disponibles y reinicia cuando sea necesario.
  2. Instalación de Controladores:
    • Automática:Windows instalará controladores básicos automáticamente.
    • Manual:Visita el sitio web del fabricante de tu notebook para descargar e instalar controladores específicos, como los de la tarjeta gráfica, chipset, red y audio.
  3. Instalación de Software Esencial:
    • Navegador web
    • Suite de oficina
    • Antivirus
    • Otros programas útiles como un lector de PDF, compresor de archivos y reproductor multimedia.
  4. Optimización del Rendimiento:
    • Ajuste del Plan de Energía:Configura un plan de energía personalizado.
    • Desactivación de Efectos Visuales:Ajusta los efectos visuales para mejorar el rendimiento.
    • Administración de Programas de Inicio:Desactiva programas innecesarios desde el Administrador de tareas.
    • Liberación de Espacio en Disco:Usa la herramienta de Liberador de espacio en disco.
  5. Eliminación de Bloatware:Desinstala aplicaciones preinstaladas no deseadas desde Configuración > Aplicaciones > Aplicaciones y características. Si prefieres, usa herramientas como CCleaner para una limpieza más exhaustiva. Ten cuidado de no eliminar aplicaciones importantes.
  6. Reinstalación de Aplicaciones Útiles:Si eliminaste aplicaciones útiles por error, reinstálalas desde el sitio web del fabricante.
